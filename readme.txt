== Description ==
Wordpress Plugin for integrate facebook commenting system on your site.

= Features =
* Facebook Commenting System for wordpress site.

== Installation ==
1. Install `SM Facebook Comments` from https://wordpress.org/plugins/sm-facebook-comments.
2. OR Upload the entire `sm-facebook-comments` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

After activate you will see comment integrated on your site post single page.
